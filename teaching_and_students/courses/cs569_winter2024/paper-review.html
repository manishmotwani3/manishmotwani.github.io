Overview

The goal of this assignment is to write a review of the paper you decide to present during class.

See these notes by John Ousterhout on writing reviews. Focus on the positive: these are classics for a reason!

Here are two example reviews for the paper coz.pdf: 

  1. The first is a real conference review.
  2. The second is a real student review for this class that you should use a model.

Your reviews must address the following points:

  • Summary: (at least two paragraphs) What is the problem that this work addressed? What are the big ideas / key insights / technical contributions
  • Significance/Contributions:  What is the practical significance of these results today? How likely are the assumptions / context this work was based on going to change in the future?
  • Discussion Points: Include at least three discussion points that you found interesting and were brought up in the class or Canvas discussions.

 

Deliverables

Each group should upload 1 pdf file (<paper-id>_review.pdf) via Canvas. There is no right length minimum or maximum:  you should write enough to describe the Summary, Significance/Contributions, and Discussion Points.
